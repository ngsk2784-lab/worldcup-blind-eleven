## 데이터 조달 현실성 (모든 컨셉 공통 전제)

**심사자가 "별도 키 없이" 확인 가능해야 함 → 라이브 키 API 금지. 정적 번들이 정답.**

- **StatsBomb Open Data** (GitHub, 무료 JSON): FIFA 남자 월드컵 2018·2022, 여자 월드컵 2019·2023의 **이벤트 단위 데이터**(슛/패스/xG/드리블 좌표, 슛 freeze-frame, 위치). 우리가 미리 다운로드·집계해 정적 JSON으로 번들 → 심사자 키 불필요. **핵심 자산.**
- **역대 스쿼드·경기결과**(Wikipedia/Kaggle 공개, 1930~2022): 명단·득점·교체 타임스탬프 등.
- **라이선스 주의**: StatsBomb은 출처표기(attribution) 필수, 비상업 용도. 해커톤 상금은 비상업으로 간주 가능하나 **선수 사진·팀 로고는 IP 리스크 → 실명(사실)만 쓰고 실루엣/도형으로 대체.**

## 후보 1 — "그 순간의 감독" (결정의 카드)

- **피치**: 실제 명경기가 흐르다 결정적 순간(60분 0-1, 부상, PK)에서 멈추고, 감독으로서 카드로 결정 → 실제 역사와 대조.
- **코어 인터랙션**: 2D 타임라인 리플레이 → 분기점에서 정지 → "직감 vs 데이터" 결정 카드 선택 → 실제 그때 무슨 일이 벌어졌는지 공개.
- **감동 포인트**: "당신은 그를 뺐지만, 실제 감독은 남겼고 그가 결승골을 넣었다" — 역사와의 대조.
- **데이터**: StatsBomb 이벤트(교체·득점 타임스탬프), 특정 명경기 5~8개 큐레이션.
- **참신성/완성도 리스크**: 예시 3번(what-if 시뮬)과 근접 → **참신성 리스크 상**. 분기 콘텐츠 수작업 큐레이션 부담.
- **3주 난이도**: 중

## 후보 2 — "블라인드 일레븐" (익명 스카우팅 드래프트) ★추천

- **피치**: 이름·국적·유명세를 전부 가린 채, 실제 월드컵 선수들의 히트맵·스탯만 보고 나만의 베스트11을 뽑는다. 완성 후 정체 공개.
- **코어 인터랙션**: 익명 카드(스파이더차트·히트맵·핵심스탯만) 비교 → 예산/포지션 제약 하에 11칸 보드로 드래그 → "정체 공개" → 실제 그 대회 성적과 대조.
- **감동 포인트**: 편견을 지우고 데이터로만 판단했을 때의 반전("무명 스탯인 줄 알았는데 메시였다 / 스타인 줄 알았는데…"). 감독의 안목을 시험당하는 몰입.
- **데이터**: StatsBomb 선수별 이벤트 집계(패스/슛/xG/드리블/수비액션)·히트맵. 우리가 사전 집계해 번들. 실명 매핑은 공개 사실.
- **참신성/완성도 리스크**: 예시에 전혀 없음 → **참신성 상**. 리스크는 히트맵·집계 사전 가공량(우리가 오프라인 처리하면 런타임 안정).
- **3주 난이도**: 중 (실시간 엔진 불필요 = 완성도 확보 유리)

## 후보 3 — "90분 지휘" (실시간 리듬 지휘)

- **피치**: 실제 경기가 2D로 리플레이되는 동안, 결정적 타이밍마다 압박·라인·전술 스위치를 발동해 모멘텀을 바꾸는 실시간 지휘.
- **코어 인터랙션**: minimap 위 실제 이벤트 흐름 + 타이밍 맞춘 지시 발동 + 모멘텀 게이지.
- **감동 포인트**: 실제 경기의 긴장을 손끝으로 지휘하는 손맛.
- **데이터**: StatsBomb 이벤트 시퀀스(패스맵·슛).
- **참신성/완성도 리스크**: "실시간 지시"가 예시 4번과 겹쳐 참신성 중. **실시간 엔진+밸런싱 = 완성도 리스크 상**(3주 내 버그 위험 최고).
- **3주 난이도**: 상

## 후보 4 — "23인의 무게" (명단 발표 드라마)

- **피치**: 월드컵 본선 명단 감독이 되어 수십 명 후보 풀에서 최종 26인을 추려 발표한다. 스타를 자르는 고통.
- **코어 인터랙션**: 후보 카드 → 최종명단 보드 드래그, 포지션/폼/부상 밸런스, 컷 할 때마다 "탈락 통보" 연출 → 발표 후 실제 명단·성적 대조.
- **감동 포인트**: "누구를 버릴 것인가" — 감독의 인간적 고뇌. 실제 논란된 컷과 내 선택 대조.
- **데이터**: 실제 월드컵 예비/최종 스쿼드(공개), 선수 스탯. **데이터 단순 = 안정적**.
- **참신성/완성도 리스크**: 명단 드라마는 예시에 없어 참신성 상, 감정선 강함. 리스크는 "동적 인터랙션"이 리스트 빌딩처럼 얕게 보일 우려.
- **3주 난이도**: 중하 (가장 안전)

## 추천: **후보 2 — 블라인드 일레븐** (차점 후보 4)

1. **참신성 최상**: 예시 4종 어디에도 없는 접근. "데이터로만 판단→정체 공개"는 심사위원이 처음 보는 훅. 참신성 30점 배점에서 가장 유리.
2. **완성도 확보 쉬움**: 실시간 엔진이 없어 3주 내 버그 없이 "안정적 작동"(완성도 기준 원문) 달성 가능. 데이터는 오프라인 사전집계 → 런타임 리스크 낮음.
3. **필수요건 정면 충족**: 카드 드래그·비교·배치가 명확한 "동적 인터랙션"이자 "감독의 안목" 몰입. 데이터 활용이 화면에 직접 드러나 데이터-인터랙션 일관성 강함.

> 후보 4(명단 드라마)는 감동·안정성이 대등해 **폴백 1순위**. 후보 2의 여운 단계에 4의 "실제 결과 대조" 연출을 흡수해 강점을 합쳤다. (두 컨셉 융합은 범위 폭주 위험 → 2를 코어로 고정하고 4 요소는 "여운"에만 차용.)

## 추천 컨셉의 감정 곡선 (감동 25점 공략 설계)

- **진입 (Hook)**: 화면에 익명 카드 몇 장. "이름을 지웠습니다. 당신의 눈을 믿으세요." — 편견을 벗기는 선언으로 감독 역할 부여. 3초 안에 무엇을 하는 게임인지 체감.
- **몰입 (Flow)**: 히트맵·스파이더차트를 비교하며 11명을 채우는 퍼즐. 예산·포지션 제약이 "감독의 고뇌"를 만든다. 카드 뒤집기·드래그의 촉각 피드백(주스)으로 조작 직관성 확보 → 감동 배점의 "조작 직관성" 항목 직격.
- **클라이맥스 (Reveal)**: "정체 공개" 버튼. 11명의 정체가 하나씩 드라마틱하게 뒤집힌다. 반전(무명↔레전드)이 터지는 순간이 이 서비스의 심장.
- **여운 (Resonance)**: 내가 뽑은 XI가 실제 그 월드컵에서 어떻게 됐는지(후보4 요소 차용) + "안목 점수" + 공유 카드. 심사위원이 스스로 한 번 더 하고 싶게 만드는 리플레이 훅.
